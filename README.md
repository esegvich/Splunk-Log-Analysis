# Splunk Log Analysis Projects

A collection of log analysis projects performed using Splunk.

## Project List

- **ssh-analysis/** — SSH authentication log investigation  
- **http-analysis/** — HTTP traffic analysis  
- **brute-force-investigation/** — Brute-force attack detection  
- **unauthorized-access-investigation/** — Suspicious access investigation  

Each project includes:
- SPL queries  
- Screenshots  
- JSON logs  
- Detailed findings
